# Backend Naming Conventions

## API Host 
{service_name}-{environment}.by-catze.xyz

e.g.

- yooldo-dev.by-catze.xyz
- yooldo-prod.by-catze.xyz
- cybergalz-dev.by-catze.xyz



## AWS RDS Naming
catze-{environment}-db

e.g.

- catze-cbt-db
- catze-dev-db
- catze-prod-db
- catze-qa-db
- catze-stage-db

### Database naming
{service_name}

e.g.

- yooldo
- cybergalz
- backoffice



e.g.
- cybergalz-dev
